## 🔴 Critical Findings

### [CR-001] Title

**Category**: [Dead Code | Quality & SOLID | Config | Specs | Tests]
**Location**: `path/to/file.php:LINE`

**Description**: What is wrong and why it matters.

**Justification**: Reference to the standard, spec, or principle violated (e.g., "Violates PRD §3.2 — CVSS calculation must use CVSS v3.1 formula").

**Example**:

```php
// Current code (problematic)
$score = $base * 1.5; // Hardcoded multiplier

// In another file, different calculation:
$score = $base * 2.0; // Conflicting multiplier
```

**Recommended Action**: Unify the calculation → use `task-planner` to plan the fix, then `implement-task`.
