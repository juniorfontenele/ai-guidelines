# UI/UX Design Documentation

This directory contains UI/UX design specifications. Generate them using:

```
/generate-ui-design
```

The skill reads your PRD and architecture docs to produce design specs covering:

- **DESIGN_SYSTEM.md** — Colors, typography, spacing, theming
- **COMPONENTS.md** — Component library and patterns
- **PAGE_INVENTORY.md** — All pages/screens with specifications
- **INTERACTIONS.md** — Animations, transitions, user interactions
- **RESPONSIVE.md** — Responsive design strategy
- **REFERENCES.md** — External design references and inspiration

> **Prerequisite**: Complete `docs/PRD.md` and `docs/architecture/` first.
